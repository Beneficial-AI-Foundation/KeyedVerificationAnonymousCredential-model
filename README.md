# KeyedVerificationAnonymousCredential-verify

A Lean 4 formalization of the keyed-verification anonymous credential (KVAC) scheme with group-element attributes, the verifiable encryption it enables, and the Signal Private Group System built on top, after [Chase, Perrin and Zaverucha (IACR ePrint 2019/1416)](https://eprint.iacr.org/2019/1416).
